## ☢ Create A Global State

Your Global State is just an object.
You Provide this state to your `<App/>` by wrapping the `<App/>` with `<Provider />`

```js
import {Provider} from 'radiactive-store'

const state = {
  todos: []
};

ReactDOM.render(
  <Provider state={state}> <App /> </Provider>,
  root,
);
```
